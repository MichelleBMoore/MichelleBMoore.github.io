Is the New and Improved: Courier New!

This typeface was originally designed in the 1950's by Howard Kettler for IBM's typewriters. But the new upgrade to the font was done by Lloyd Springer. But wait! There is more. They didn't secure it legally so it soon became a standard font for all typewriters in the industry. Kettler's font is a monospaced serif font. With the font being monospaced, the letters will be evenly spaced. The computer adopted this font when they needed a font that could be evenly spaced. Plus this typeface is the industry standard for all screenplays.
